# DOM Tree and Data Structures

The **dom tree** is made up of **nodes**.

There are various types of nodes:
- Document
- DocumentType
- DocumentFragment
- Element
- Text
- ProcessingInstruction
- Comment

## Node data structure
> Each node has an associated node document, set upon creation

Node
- NodeType
- ParentNode
- NodeName
- ChildNodes
